Политика конфиденциальности

Последнее обновление: 29 марта 2026 г.

1. Общие положения

Настоящая политика конфиденциальности регулирует использование приложения «Свой чат», разработанного PROGRAMISTIK. Используя приложение, вы соглашаетесь с условиями данной политики.

2. Собираемые данные

Приложение может собирать следующие данные:

  • Имя пользователя и пароль для входа в систему
  • Текстовые сообщения в рамках чатов
  • Данные об устройстве (модель, версия ОС)

3. Использование данных

Собранные данные используются исключительно для:

  • Обеспечения работы сервиса обмена сообщениями
  • Идентификации пользователей
  • Улучшения качества приложения

4. Защита данных

Все сообщения в приложении зашифрованы. Мы не передаём ваши данные третьим лицам.

5. Разрешения

Приложение запрашивает следующие разрешения:

  • Камера — для видеозвонков
  • Микрофон — для голосовых звонков
  • Интернет — для работы приложения
